<h2 align="center">Introduction</h2>
<div align="center">
  这是一个SPA+SSR网站开发的构筑工具
</div>

<div align="center"></div>
<h2 align="center">Install</h2>

```bash
npm install @cyber-chimera/spa-build-tools --save-dev
```

<h2 align="center">Usage</h2>

### 相关命令:

开发模式启动命令:

```bash
spa-build-tools dev
```

生产环境编译命令:

```bash
spa-build-tools build
```
